https://hub.docker.com/_/influxdb

docker pull influxdb

I am using a local config file found in C:/docker/influxDB/config and 
the data is stored loacally as well in C:/docker/influxDB/data

All of this is specified in the docker-compose.yml file found at C:\docker\influxdb

Run it:
docker-compose up -d

Show me the logs:
docker-compose logs

Stop it:
docker-compose stop
docker-compose rm

This is how to run the client agasint the influxdb01 container
sudo docker exec -it influxdb01 influx -username tcmadmin -password tcmadmin

some client commands
SHOW DATABASES  //returns all the databases
use db0  //this allows us to select the database db0
select * from node_status   //run some queries
select mean(cpu_usage) from node_status  //more queires
select * from node_status limit 3  //limits the number of return points
